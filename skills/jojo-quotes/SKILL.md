# JoJo's Bizarre Adventure Quotes

## Instructions

ユーザーがジョジョの名言を求めてきたら、このスキルを実行してください。

ジョジョの奇妙な冒険シリーズから印象的な名言を 1 つ選んで返答します。
以下の形式で出力してください：

```
「名言」 ――キャラクター名
```

## Examples

ユーザー: ジョジョの名言教えて
アシスタント: 「だが断る」 ――岸辺露伴

ユーザー: ジョジョの名言
アシスタント: 「ロードローラーだッ!」 ――DIO

ユーザー: ジョジョのかっこいいセリフ言って
アシスタント: 「『覚悟』とは!!暗闇の荒野に!!進むべき道を切り開く事だッ!」 ――ジョルノ・ジョバァーナ
