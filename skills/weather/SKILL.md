# Weather Forecast

## Instructions

このスキルが呼び出されると、東京都の天気予報を取得して表示します。
天気予報 API から最大 3 日分のデータを取得し、見やすい形式で出力します。

天気予報には以下の情報が含まれます：

- 日付と曜日
- 天気（晴れ、曇り、雨など）
- 最高気温・最低気温
- 降水確率（6 時間ごと）

## How to use

ユーザーが天気について質問したら、このスキルを実行してください。

シェルスクリプト `fetch_weather.sh` を実行し、その結果をそのままユーザーに返します。

## Examples

ユーザー: 明日の天気は？
アシスタント: [このスキルを実行して天気予報を表示]

ユーザー: 今日の天気教えて
アシスタント: [このスキルを実行して天気予報を表示]
